External Plugins

Note

You simply need to install a compatible plugin and Breaker will automatically hook onto it!

  • MMOItems:
    Using the MMOStatState or NBTState you can check for the NBT of a unique MMOItem or the players specific stat level.
  • MMOCore:
    MMOCore provides a few states related to features like level, experience, professions, classes and more.
  • MythicLib:
    MythicLib provides a simple mmostat state to check against a players specific MMO Stat.
  • TechTree:
    TechTree provides a simple techentry state to check if a player has unlocked a specific entry.
  • Vault:
    Vault provides a simple money state to check against a players vault balance.
  • WorldGuard:
    WG provides not only a Region State, but also allows you to use the “Region Rules”. For more information about this, please see Breaker’s config.yml file.
