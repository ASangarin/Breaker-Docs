States

Configuring the States

States are conditions that can change the blocks Hardness when they’re met. There are a ton of States available, both provided by breaker and by external plugins.

Attention

See Configuration to understand how states are configured!
This page is a simple reference page to quickly find the state you’re looking for.

Available States

Built-In States:

  • effect

  • enchant

  • exp (Experience)

  • helditem

  • air

  • water

  • sneaking

  • time

  • unbreakable

  • rain

  • world

  • permission

  • mmostat

  • nbtbool (NBT Boolean)

  • nbtstr (NBT String)

  • nbtval (NBT Value)

MMOCore States:

  • mmoattribute

  • mmoclass

  • mmoexp

  • mmopoint

  • mmoprof (MMO Profession)

  • mmoresource

MythicMobs States:

  • mmcast (MM “Can Cast?”)

  • mmvar (MM Variable)

TechTree States:

  • techentry

Vault States:

  • money

WorldGuard States:

  • region
