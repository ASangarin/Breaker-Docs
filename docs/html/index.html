What is Breaker?

Breaker is a free, yet powerful Spigot plugin which lets you set block-breaking speeds for blocks to whatever you’d like. Want to mine Obsidian quicker? Breaker. Want to make certain wood types harder than others? Breaker can do it. Breaker also features systems such as conditions (called states) for reducing break speed and triggers for making things happen. This documentation contains information about everything that Breaker has to offer.

Get Breaker at SpigotMC.
